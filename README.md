# 🍔 Snack Hub

Snack Hub is a modern and responsive food ordering website built using HTML, CSS, and JavaScript.

This project was created to practice frontend web development and improve UI/UX design skills.

---

# 🚀 Features

- Responsive Navbar
- Mobile Hamburger Menu
- Premium Hero Section
- Product Cards
- Add to Cart Functionality
- Cart Sidebar
- Checkout Popup
- Search Function
- Category Filters
- Scroll Reveal Animations
- Responsive Design for Mobile & Desktop
- Smooth UI Effects

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

---

# 📱 Responsive Design

The website works smoothly on:

- Mobile Phones
- Tablets
- Laptops
- Desktop Screens

---

# 🎯 Purpose of Project

This project was built as:

- A frontend practice project
- Portfolio project
- Mini project for learning
- Freelancing showcase project
